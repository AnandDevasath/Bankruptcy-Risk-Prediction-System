# Bankruptcy Risk Prediction System

🔍 Bankruptcy Risk Prediction System

This project aims to predict the likelihood of financial bankruptcy using machine learning techniques. It involves data preprocessing, exploratory data analysis (EDA), feature engineering, and model building using classification algorithms. Key financial indicators are analyzed to identify risk patterns and improve prediction accuracy.

The model is evaluated using performance metrics such as accuracy, precision, recall, and confusion matrix. The final model is saved as a .pkl file and integrated into a Streamlit web application for real-time prediction and decision support.

🚀 Key Features:

Data cleaning and preprocessing, 
Exploratory Data Analysis (EDA), 
Feature importance analysis, 
Model training and evaluation, 
Streamlit-based deployment .

🛠 Tech Stack:

Python, 
Pandas, NumPy, 
Machine Learning, 
Scikit-learn, 
Matplotlib, Seaborn, 
Streamlit .

📌 Objective:

To build an efficient predictive system that helps identify potential bankruptcy risks and supports better financial decision-making.
